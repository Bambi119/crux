# 9. 맵 & 스테이지 구조

> 작성: 2026-04-16 — 초판
> 관련 문서: `03_core_gameplay_loop.md` (루프) · `06_combat_system.md` (전투) · `05_vehicle_system.md` (전차)
> **상위 결정**: 무분기 일직선 스토리 · 미디엄 캠페인 · Banner Saga 라이트 (이동 캠프 + 월드맵)

## 0. 개요

- **월드맵 + 이동하는 캠프** 모델 (Banner Saga 라이트 버전)
- **총 노드 51개** / 전투 22 + 대화 15 + 거점 8 + 휴식 6
- **예상 캠페인 길이**: 10~12시간 (전투 평균 25분 × 22 + 비전투 노드 + 캠프 시간)
- **첫 빌드 단순화**: 일직선 노드, 자원·식량 시스템 없음, 사기 1축 + 자금만

스토리 톤: 세기말 바이오코어 지배 세계, 야욕·전쟁이 끝난 자리에 남은 권력 잔당과 약탈자 사이를 헤치고 **유토피아를 찾아 떠나는 유랑단의 여정**. 무거운 서사·인물 관계 중심.

## 1. 월드맵 구조

### 1.1 노드 개념

월드맵은 **노드(Node) 그래프**로 구성된다. 노드는 월드맵 위의 한 지점 = 하나의 이벤트 단위.

| 용어 | 정의 | 예 |
|---|---|---|
| 노드 (Node) | 월드맵의 한 지점 = 한 이벤트 단위 | 화면 위 작은 동그라미 |
| 스테이지 (Stage) | 전술 전투 미션. **노드의 한 종류** | 전투 노드 클릭 → 전투 진입 |
| 씬 (Scene) | Unity 기술 단위 | 전투 씬·캠프 씬·대화 씬·월드맵 씬 등 4~5개 고정 |

플레이어는 월드맵에서 다음 노드로 한 걸음씩 진행. 노드 종류에 따라 다른 화면 진입 → 처리 후 월드맵 복귀.

### 1.2 노드 종류 및 분포

캠페인 1회분 51 노드 분포:

| 종류 | 수 | 역할 |
|---|---|---|
| **전투 노드** | 22 | 메인 전술 미션 (스테이지 = 이것) |
| **대화 노드** | 15 | 메인 스토리 12 + 캐릭터 관계 3 |
| **거점 노드 (캠프)** | 8 | 상점·편성·정비·스킬·회식 등 풀 메뉴 |
| **휴식 노드** | 6 | 짧은 사기 회복·여정 묘사 |
| 합계 | **51** | |

### 1.3 첫 빌드 단순화 (1~2개월 목표)

- 일직선 노드 그래프 (서→동 또는 위→아래 한 방향)
- 노드 분기 없음. 한 노드 끝 → 다음 노드 자동 활성화
- 노드 간 이동 자체에 자원·시간 소모 없음 (단순 클릭으로 진행)
- 데모용으로는 5~10 노드만 채워서 한 바퀴 검증

### 1.4 향후 확장 (스토리 확정 후)

- 지역별 월드맵 디자인 (사막·폐허 도시·숲·해안 등 4~6 지역)
- 노드 위치가 시각적으로 의미 있는 지도 위에 배치
- 후반에 분기 노드 도입 가능 (선택지에 따른 짧은 우회 경로)

## 2. 노드 종류별 상세

### 2.1 전투 노드 (스테이지) — 22개

진입 시 전투 씬 (`TacticalScene`) 로드. 평균 플레이 시간 **20~30분**.

#### 미션 종류 5종 분포

| 종류 | 승리 조건 | 분포 | 비고 |
|---|---|---|---|
| **섬멸** | 적 전차 전멸 | 8/22 | 가장 흔함, 기본 형태 |
| **방어** | 지정 지점 N턴 사수 | 4/22 | 정적 배치, 사기 압박 |
| **호위** | NPC/차량 안전 보호 | 4/22 | 동적 목표, 난이도 높음 |
| **탈출** | 지정 노드 도달 | 3/22 | 적의 추격 + 시간 압박 |
| **점령** | 지점 점거 N턴 | 3/22 | 공방 균형, 위치 전환 |

#### 난이도 배치
- 캠페인 후반으로 갈수록 적 구성 강화·미션 복잡도 증가
- 플레이 시간 부족분은 후반 미션의 난이도·길이로 보완 (당사자 결정)

### 2.2 대화 노드 — 15개

진입 시 대화 씬 또는 컷씬 자동 재생. 무거운 스토리 진행 + 캐릭터 관계 단계 변화.

| 종류 | 수 | 트리거 |
|---|---|---|
| 메인 스토리 진행 | 12 | 노드 진입 시 자동 (스킵 가능, 1회만) |
| 캐릭터 관계 이벤트 | 3 | 특정 캐릭터 관계도 임계값 도달 시 자동 |

설계 원칙:
- 대화 노드는 **의미 있는 스토리 전환 지점**에만 배치
- 짧은 대사·소소한 농담은 노드로 만들지 않고 **전투 전후 대사 / 캠프 NPC 클릭 대화**로 처리
- 무분기 일직선이지만 **선택지에 따른 약간의 결과 차이** 가능 (관계도 변화, 사기 변화 등)

### 2.3 거점 노드 (캠프) — 8개

진입 시 캠프 씬 (`Hangar` 통합) 로드. 풀 메뉴 진입.

#### 능동형 액션 (플레이어가 선택)
- **부대 편성** — 전차 5대 슬롯 + 파츠 교체
- **정비** — 차체 수리, 모듈 회복
- **상점** — 연료·탄약·파츠 구매 (자금 소모)
- **스킬 카드 선택** — 전투에서 쌓인 마크 각성 큐 처리
- **회식 개최** — 사기 +10~20 (자금 소모)
- **NPC 대화** — 캠프 인물 클릭, 관계도 + 사이드 정보

#### 수동형 이벤트 (자동 발생)
- **추모식** — 전사자 발생 후 첫 캠프 진입 시 자동, 스킵 불가, 사기 약간 회복
- **메인 스토리 컷씬** — 특정 거점 노드에 고정 이벤트
- **시설 해금 알림** — 진행도 임계 도달 시 자동

### 2.4 휴식 노드 — 6개

짧은 분량의 자동 이벤트 + 소량 사기 회복. 캠프보다 가볍고, 대화보다 일상적.

예시:
- 길가에 차량 정지, 짧은 잡담 컷
- 야영 묘사, 다음 날 출발
- 길동무 조우 (1회성, 짧은 대사)

전투·캠프·대화 사이 **호흡**을 만드는 노드. 게임 페이스 조절용.

## 3. 캠프 시설 점진 확장

거점 노드의 시설은 캠페인 진행에 따라 단계적으로 해금된다. 자금이 아닌 **스토리 진행도** 기준.

| 단계 | 해금 시점 | 추가 시설 | 누적 시설 |
|---|---|---|---|
| **초반** | 캠페인 시작 | 상점 (연료·탄약·기본 파츠) · 게러지 (소) | 2종 |
| **중반 1** | 전투 8~10회 후 | 정비소 (수리·파츠 업그레이드) · 훈련소 (스킬·경험치 가속) | 4종 |
| **중반 2** | 전투 15~17회 후 | (자동 확장 — 게러지 중형) | 4종 |
| **후반** | 전투 18~20회 후 | 병원 (부상 치료 가속) · 술집 (사기 이벤트 다양화·추가 회식) | 6종 |

---

## 4. 부대 출격 / 보관 슬롯 확장

게러지 확장에 연동된 슬롯 수.

| 단계 | 출격 (전투 출전) | 보관 (게러지 수용) |
|---|---|---|
| 초기 | 3 | 5 |
| 중반 1 | 4 | 7 |
| 중반 2 | 5 | 8 |
| 후반 | 5 (최대) | 10 |

**출격 5 / 보관 10**: 출격은 작전 한 판당 5대 고정 (한 판 20~30분 보장), 보관은 여유 있게 10대까지 모아두고 편성 시 5대 선별. 전차 빌드 다양성 + 편성 의사결정의 재미 보장.

로시난테는 항상 보관 1슬롯 점유 (고정), 출격 시 의무 선택 아님 (단, 스토리 일부 미션에 강제 출전).

## 5. 자원 관리 — 단순화

식량·연료·시간 자원 시스템 **없음**. 첫 빌드와 정식 출시 모두 동일.

| 자원 | 사용 처 | 회복 방법 |
|---|---|---|
| **사기 (Morale)** | 전투 명중률·AP·반응 사격에 영향 | 회식·추모식·야영 대화·시간 (자연 회복은 없음, 이벤트로만) |
| **자금** | 상점 구매 (연료·탄약·파츠), 회식 개최, 시설 일부 | 전투 승리 시 보상, 노획 파츠 매각 |

**없는 자원**: 식량 (유랑단 생존 컨셉 포기) · 연료 (이동 자원 아님, 차량용 소비재로만 상점 구매) · 시간 (캠페인 일수 카운터 없음)

---

## 6. 자동 진행 항목 (플레이어 액션 아님)

| 항목 | 진행 방식 |
|---|---|
| **부상자 치료** | 자동 시간 기반 (전투 회수 단위). **의사 캐릭터** 합류 시 가속 |
| **사기 자연 회복** | **없음**. 회식·추모식·야영 대화 이벤트로만 |
| **노드 간 이동** | 자동, 자원·시간 소모 없음 (첫 빌드 기준) |
| **시설 확장** | 진행도 임계 자동 해금. 자금 비용 없음 (단순화) |
| **마크 누적** | 전투 후 자동 정산. 임계 도달 시 캠프에서 카드 선택 큐 적재 |

---

## 7. 자원 정산 흐름

```
전투 종료
  ↓
[전투 결과 화면 — 즉시 팝업]
  - 격파 수, 노획 파츠 리스트, 사기 변화, 부상 정보
  - 자금 보상 표시
  ↓
[월드맵 복귀]
  ↓
[다음 노드가 거점이면 → 캠프 진입]
  - 인벤토리에서 노획 파츠 확인
  - 편성 변경 / 정비 / 상점 / 스킬 카드 선택
  ↓
[다음 노드 진행]
```

## 8. 첫 빌드 (1~2개월 목표) 검증 단위

루프 폐곡 검증을 위한 최소형:

- 일직선 월드맵 1개 (5~10 노드만)
- 전투 노드 3개 + 캠프 노드 1~2개 + 대화 노드 1~2개
- 거점 시설은 **상점·편성** 2종만
- 출격 슬롯 3, 보관 5 (초기 단계만)
- 미션 종류는 섬멸 1종만 (다양성은 후순위)
- 사기 이벤트는 회식만
- **목표**: "한 판 처음부터 끝까지 가는 빌드" → 외부 테스터에게 뿌려 재미 검증

## 9. 미션 평균 시간 목표

| 미션 종류 | 목표 시간 |
|---|---|
| 섬멸 | 20~25분 |
| 방어 | 25~30분 |
| 호위 | 25~30분 |
| 탈출 | 15~20분 |
| 점령 | 25~30분 |
| **평균** | **20~30분** |

기준: Front Mission · XCOM 2 표준. 한 판 30분 넘으면 퇴근 후 1판 부담. 25분 평균 + 변동 ±5분이 적정.

## 10. 향후 결정 필요 항목

- 미션별 구체 시나리오 (지형·적 구성·승리 조건 변형) — 22편 분량
- 대화 노드 구체 시나리오 (메인 스토리 12편 + 캐릭터 3편)
- 캠프 NPC 대화 풀 (여러 라운드 분량의 대사)
- 자금 경제 수치 (상점 가격·회식 비용·보상 곡선)
- 의사 캐릭터 합류 시점 (캠페인 어느 지점)
- 시설 해금 자금 비용 도입 여부 (현재 자동 해금)
- 사기 자연 회복 도입 여부 (현재 이벤트만)
- 노드 간 이동에 시간/자원 소모 도입 여부 (현재 무료)
- 분기 노드 후반 도입 여부 (현재 무분기)
