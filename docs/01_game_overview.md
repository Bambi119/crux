# 1. 게임 개요

- **프로젝트명**: CRUX (철갑을 두른 당나귀)
- **장르**: 실시간 탑뷰 서바이벌 액션 + 전차 수집/육성/조립
- **핵심 공식**: 뱀서 + 육성 + 조립
- **플랫폼**: Steam (패키지 판매, 스토리 엔딩 있음)
- **엔진**: Unity 6.3 LTS (C#)
- **그래픽**: 2D 탑다운, 현실적 도트 (픽셀아트), 무거운 톤
- **시점**: 탑다운 (위에서 내려다보는 시점)

## 레퍼런스

| 게임 | 참고 요소 |
|------|----------|
| Vampire Survivors | 실시간 웨이브 서바이벌 루프, 자동/반자동 공격 |
| Front Mission | 파츠 조합 → 플레이 스타일 변화, 중량/출력 시스템 |
| RIOT: Civil Unrest | 아트 톤 (리얼리즘 도트) |
| Pathway | 아트 톤 (군사/탐험) |

## 핵심 차별점

1. **전차 1대를 깊게 커스터마이징** — 로그라이크식 랜덤이 아니라, 영구적으로 파츠를 모으고 조립
2. **장착 무기에 따라 플레이가 바뀜** — 속사포, 대구경포, 기관총, 로켓 등
3. **중량/출력 트레이드오프** — 무거운 장갑 = 느린 기동, 강한 엔진 = 빠르지만 연료 소모
4. **스토리 기반 스테이지 진행** — 로그라이크 무한반복이 아닌, 엔딩이 있는 여정

## 시놉시스

바이오 코어에 감염된 기계 괴물들을 뚫고 안식처를 찾아 떠나는 소년들, 그리고 낡은 전차 '로시난테'의 이야기.
