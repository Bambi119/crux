# 1. 게임 개요

> 개정: 2026-04-15 — 실시간 서바이벌 방향 폐기. 턴제 Hex 전술 + 부대 운용으로 재정립.

## 기본 정보

- **프로젝트명**: CRUX (철갑을 두른 당나귀)
- **장르**: 턴제 Hex 전술 전차 전투 + 부대 편성 + 파츠 조합/육성
- **핵심 공식**: **XCOM + Front Mission + 포스트아포 서사**
- **플랫폼**: Steam (패키지 판매, 스토리 엔딩 있음)
- **엔진**: Unity 6.3 LTS (C#)
- **그래픽**: 2D 탑다운, 현실적 도트 (픽셀아트), 무거운 톤
- **시점**: 탑다운 (위에서 내려다보는 시점)
- **그리드**: 육각형 (flat-top hex, odd-q offset)

## 핵심 차별점

1. **전차 = 캐릭터** — 단순한 탈것이 아닌 5인 승무원 + 파츠의 총체. 각 전차는 고유 서사와 역할을 가진다
2. **격파 방식이 경제 결정** — 어떻게 적을 파괴했는가(탄약고 유폭 / 관통 / 승무원 사살 / 승무원 전원 사망)가 노획 품질을 직접 결정. 플레이 스타일 = 빌드 경제
3. **호환성 3중 체계** — 하중·출력·규격 세 축이 모두 통과해야 파츠 조립 성립. 단순 슬롯 매칭이 아닌 실제 공학적 긴장감
4. **마크 숙련도 + 블라인드 스킬** — 성장 타이밍은 예측 가능하지만 획득 스킬은 블라인드. 반복 플레이의 재미
5. **로시난테 — 서사적 고정 전차** — 중형전차 고정, 폐기 불가. 5인 승무원 선호와 의도적 불일치에서 시작해 정비 진행이 곧 각성 서사

## 시놉시스

> 상세: `docs/02 세계관 & 스토리` v2.0

가상 아프리카 독재국의 **격리 구역 케마르**. 의료용이었던 **바이오 코어(Bio-Core)** 기술이 서방 군사 제국에 의해 무기로 전용된 세계. 시타 박사의 발명은 강화 병사와 자율 무기가 되어 인류 자신을 향하고, 격리구의 주민들은 정부군·해방 민병대·민간 용병의 3중 폭력 속에서 살아간다.

격리구에서 자란 소년 **아스트라**(16), 동생 **이리스**(13), 맏언니 **프리테나**(19), **리리드**(15), **그린**(17)은 폐허 군수 창고에서 **자율 시스템이 없어 바이오 코어에 감염되지 않은 재래식 중형전차**를 발견, 직접 손으로 복원해 **로시난테**라 이름 붙인다.

플레이어는 로시난테와 5인 승무원으로 첫 출격을 시작해, 전투·노획·정비를 반복하며 **새 전차·새 승무원을 부대에 합류**시킨다. 1막 용병의 학살을 피해 도망치던 여정은, 시타 박사를 만나 그의 비밀 — 전 세계 바이오 코어를 일시에 무력화하는 **코드키** — 을 알게 되면서 **위성 네트워크 시설로의 침투 임무**로 전환된다.

"철갑을 두른 당나귀"는 초라한 시작에서 출발하되 영웅이 되지 않는 이야기다. 평범한 어른들의 희생 위에서 아이들은 살아남고, 마지막에는 세계를 조용히 뒤집은 뒤 존재하지 않는 사람이 되어 한적한 곳으로 떠난다.
