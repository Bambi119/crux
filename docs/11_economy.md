# 11. 경제 시스템

> 초안: 2026-04-21
> 근거 문서: docs/09 §5 (2축 경제 원칙) · docs/05 §4-5 (파츠·노획) · docs/04 §6.1 (사기) · docs/03 (게임 루프)
>
> **원칙**: 2-axis 경제(자금 + 사기) + 파츠는 시장 통화로 변환되지 않는 별개 자원. 시간·식량·여행 연료는 자원이 아니다.

---

## 0. 설계 철학

1. **희소성은 중반 압박에서만 작동** — 초반 빈곤(5분 이내 체감) → 중반 긴장(빌드 갈등) → 후반 풍요(선택 폭)
2. **자금과 파츠는 분리** — 파츠는 팔 수 있으나 **살 수 없다**. 모든 의미 있는 파츠는 노획·스토리로만 입수
3. **회식은 유일한 사기 회복 수단** — 자연 회복 없음(docs/09 §6). 사기 관리를 캠프 선택에 묶는다
4. **유지비 없음** — 턴당·스테이지당 자금 감산 없음. 손실은 오직 **구매·회식 선택 시점**에 발생
5. **자금 경제는 "사고 싶은 것 > 가진 돈"을 지속 유지** — 스토어 구매는 항상 기회비용 발생해야 함

## 1. 자원 축 정의

| 축 | 범위 | 저장 위치 | 이월 | 증감 주체 |
|---|---|---|---|---|
| **자금 (Funds)** | 0 ~ ∞ (int) | ConvoyInventory | 전 캠페인 | 전투 보상·파츠 매각 (+) / 상점·회식 (−) |
| **사기 (Morale)** | 0~100 | TankCrew 단위 (전차별) | 전투 간 이월 | 전투 이벤트·회식 (+) / 부상·파손 (−) |
| **파츠 (Parts)** | 개수 | ConvoyInventory.partStash | 영구 | 노획·스토리 (+) / 분해·매각 (−) |
| **승무원 (Crew)** | 20명 풀 | Roster | 영구 | 스토리 해금 (+) / 치명상(스토리만) (−) |

**의도적 비포함**: 시간, 식량, 전략맵 연료, 턴 리밋, 부대 유지비.

## 2. 자금 수입원

### 2.1 전투 승리 보상 (기본값)

```
보상 자금 = base(미션종류) × 난이도계수 × 부가조건
```

| 미션 종류 | base 자금 | 비고 |
|---|---|---|
| 조우전(Skirmish) | 150 | 소규모 적 2~3대 |
| 거점 방어(Hold) | 250 | 파도형, 사기 페널티 큼 |
| 적진 침투(Raid) | 300 | 목표 격파·탈출 |
| 보스(Boss) | 600~1000 | 스토리 분기 |
| 정찰(Scout) | 100 | 전투 선택적 회피 가능 |

| 난이도 | 계수 |
|---|---|
| Act 1 (초반) | ×1.0 |
| Act 2 (중반) | ×1.6 |
| Act 3 (후반) | ×2.4 |

### 2.2 부가 조건 보너스 (가산)

| 조건 | 보너스 |
|---|---|
| 아군 피해 0 관통 | +20% |
| 터닝 내 목표 달성 (조기 클리어) | +15% |
| 전 적 승무원 사살 노획(§2.3과 별도) | +10% (정보 수집 가치) |
| 민간 구조물 보존 | +10% (일부 미션) |
| 특정 적 우선 처치 지령 | +25% |

### 2.3 노획 파츠 매각

스토어·캠프 "분해·매각" UI에서 등급별 정액 환산:

| 등급 | 매각가 (base) | 비고 |
|---|---|---|
| 쓰레기 | 20 | 대량 노획 시 쏠쏠 |
| 일반 | 80 | MVP 주 수입원 |
| 숙련 | 200 | 장착 대신 팔지 여부 고민 |
| 정예 | 600 | 대부분 장착이 유리 |
| 전설 | 2000 | 거의 팔지 않음(그러나 슬롯 압박 시 선택지) |

**매각가 = 상점 이론가 × 30%**. 의도적 저효율: 파츠는 **팔려고 쌓는 자원이 아니다**. 인벤토리 정리 및 긴급 자금 보충 역할.

### 2.4 스토리 고정 보상

스토리 이벤트 클리어 시 일회성 자금 + 지정 파츠. Act 경계마다 배치:
- Act 1 중반: +400 (리소스 점프)
- Act 2 중반: +1200
- Act 3 시작: +2400

### 2.5 분기 선택형 보상

스토리 분기에서 "자금 300 받기 / 숙련 엔진 1개 받기" 같은 상호 배타 선택. 빌드 방향성을 플레이어에게 묻는다.

## 3. 자금 지출처

### 3.1 상점 가격 (MVP)

#### 탄약 (소모품)

| 탄종 | 단가 (10발) | 비고 |
|---|---|---|
| AP | 40 | 표준 |
| HE | 60 | 범위 효과 프리미엄 |
| 로켓 | 120 | 제한적 |
| 소이(화재) | 100 | Phase 2 소이 시스템 연동 |

전차별 탄약고 최대치 기준 완충까지 일괄 구매. 낭비 없도록 "보유량 / 최대치" UI 표시.

#### 연료 (docs/09 향후 결정 — MVP에는 생략)

**MVP 1차 빌드는 연료 비포함**. 향후 전략맵 연료가 도입되면 단가 50/100마일 기준으로 편입.

#### 파츠 (제한 SKU)

- 상점은 **소모성·범용 파츠**만 판매
- 주포·엔진·장갑 같은 **핵심 파츠는 판매 금지** — 노획·스토리에 고정
- 판매 허용: 기본 탄약고, 보조 장비(응급 키트 등), 낮은 등급 캐터필러

| SKU | 가격 | 비고 |
|---|---|---|
| 표준 캐터필러 (Common) | 300 | 파손 복구용 |
| 응급 키트 (Aux) | 150 | 1전투 소모 |
| 연막탄 발사기 (Aux) | 400 | 1 교전 3회 |
| 조준 보조 (Aux Skilled) | 600 | 명중 +5 |
| 적외선 스코프 | 800 | 연막 관통 |

#### 부상 치료 — 무료 자동 회복 (docs/09 §6)
자금 비용 없음. 대기 턴만 소모. (향후 "빠른 치료 유료" 옵션 고려)

### 3.2 회식 (사기 회복)

| 회식 등급 | 비용 | 효과 | 사용처 |
|---|---|---|---|
| 간단한 식사 | 100 | 출격 가능 전차 전체 사기 +10 | 일반 휴식 |
| 정찬 | 300 | 전 전차 사기 +25 | 큰 전투 직전 |
| 축제 | 800 | 전 전차 사기 +50 + 다음 전투 1회 "사기충천" 고정 보장 | 스토리 분기 |

**사기 환율 기준**: 자금 4~10당 사기 1. 중반 자금 압박기에 회식은 의도적 "뼈아픈 선택"이어야 한다.

### 3.3 향후 지출처 (First Build 이후)

| 항목 | 의도 | 편입 시점 |
|---|---|---|
| 시설 해금 (정비소·훈련소·병원·술집) | Phase 2 확장 (docs/09 §8 향후) | 중반 빌드 |
| 승무원 모집 (랜덤 풀) | 캐릭터 자유도 확장 | Phase 2 |
| 빠른 부상 치료 | 중상 대기 스킵 (유료) | Phase 2 |
| 스킬 리셋 | 장착 스킬 재선택 | 밸런스 패스 |

## 4. 노획 경제 (파츠 시장)

**docs/05 §5 참조**. 경제 관점 요약:

### 4.1 파츠 가치 계층

```
노획 등급 분포 (격파 방식 의존)
├─ 쓰레기: 60~70%
├─ 일반:   20~30%
├─ 숙련:    5~10%
├─ 정예:    1~3%
└─ 전설:    <1% (특정 미션 한정)
```

플레이어 행동이 분포를 바꾼다 — "승무원 전원 사망" 루트는 정예·전설 비율 10배.

### 4.2 파츠 인플레이션 방지

- 등급 기반 **환산가 고정** (§2.3) — 매각가는 오르지 않는다
- 고등급 파츠는 **장착 압박** 설계: 정예 1 매각 = 일반 7.5 매각 수준이지만, 장착 시 빌드 전환 가치는 그 이상
- 공용 인벤토리 슬롯 상한 도달 시 폐기 강제 → **보유 피로**가 매각 유인

### 4.3 분해(환원) vs 매각

| 행동 | 반환 | 용도 |
|---|---|---|
| 매각 | 자금 (등급 고정가) | 즉시 현금 |
| 분해 | 동일 카테고리 재료 (등급 -1) | 파츠 업그레이드 (Phase 2) |

MVP에서는 분해 UI 제공, 재료 사용처는 Phase 2 업그레이드 시스템 도입 시 해금.

## 5. 사기 경제

### 5.1 사기 이벤트 (docs/04 §6.1 재정리)

| 이벤트 | 변동 | 빈도 |
|---|---|---|
| 적 격파 | +5 | 전투당 3~6회 |
| 도탄 (튕김) | +3 | 전투당 2~4회 |
| 관통 명중 | +3 | 전투당 2~3회 |
| 전차장 격려 스킬 | +10~20 | 쿨다운 3턴 |
| 승무원 부상 | −15 | 전투당 0~2회 |
| 모듈 파손 | −5 | 전투당 0~3회 |
| 측·후면 피격 | −10 | 전투당 0~2회 |
| 탄약고 근접 피격 | −20 | 전투당 0~1회 |

**예상 순변동 (평균 전투)**: +15 ~ +30 (승리 시) / −20 ~ −40 (난투 피해)

### 5.2 사기와 자금의 교환

회식(§3.2)을 통해서만 가능. **자연 회복 없음**(docs/09 §6) — 사기 방치는 성능 저하 누적.

### 5.3 공황 안전장치

전투당 1회 사기 0~24 진입 시 자동 +15 (docs/04 §6.1). 경제 관점에서 이는 **데스 스파이럴 방지 장치** — 플레이어가 회식 예산 없이 연속 패배 시 완전 무력화되지 않도록 한다.

## 6. First Build MVP 수치

**전투 3 + 캠프 1~2 + 대화 1~2** 규모(docs/09 §8)에서의 구체 균형.

### 6.1 목표 자금 곡선

| 시점 | 보유 자금 (기대치) | 비고 |
|---|---|---|
| 캠페인 시작 | 200 | 초기 밑천 |
| 1차 전투 후 | 350 | 조우전 150 + 매각 ~50 |
| 캠프 1 체류 중 | 200~450 | 회식 or 절약 선택 |
| 2차 전투 후 | 500~700 | 조우전 or 방어 |
| 3차 전투 후 (종료) | 800~1200 | 보스 600 포함 |

**의도**: 플레이어가 캠프 1에서 "회식할지 탄약 사서 3차전에 임할지" 결단해야 하는 수준의 타이트함.

### 6.2 3차 전투 보스 예산 역산

보스전 직전 사기 50~79 유지 + 탄약 AP 20발·HE 10발 확보 조건:
- 필요 지출: 탄약 80+60 = 140, 비상 회식 100 = 240
- 첫 전투 후 자금 350~400이 유지되어야 이 예산이 유지됨 → §6.1 곡선과 일치

### 6.3 파츠 흐름 (3전투)

| 전투 | 노획 기대 |
|---|---|
| 1차 (조우 2대) | 일반 2~3개, 쓰레기 1~2개 |
| 2차 (조우 3대) | 일반 3~4개, 숙련 0~1개 |
| 3차 (보스) | 정예 1개 (고정 드롭), 일반 2~3개 |

3차 전투 종료 시 **정예 파츠 1개 장착 결정**이 플레이어 첫 빌드 선택 체험의 핵심.

## 7. 경제 압박 곡선 (풀 캠페인)

### 7.1 초반 (Act 1, 전투 1~6)

- 자금 빈곤. 대부분 "기본 탄약 보충 + 최소 수리" 선에서 예산 소진
- 매 캠프 진입 시 "회식? 파츠 구매? 탄약 비축?" 중 한두 개만 가능
- 파츠 풀 얕음 — 노획 하나하나가 빌드 변경 트리거
- **체감**: 스카치, 선택의 무게감

### 7.2 중반 (Act 2, 전투 7~14)

- 자금 증가하지만 **지출 옵션도 급증** (시설 해금, 보조 장비 고급 SKU)
- 파츠 슬롯 포화 시작. 분해·매각·폐기 판단 필요
- 사기 이벤트 다발 — 고사양 전투 연속, 회식 중요도 최대
- **체감**: 선택의 폭이 넓어지되 예산은 여전히 부족

### 7.3 후반 (Act 3, 전투 15~22)

- 자금 여유. 탄약·회식은 기본 구매
- 파츠는 "최적 빌드 조립" 단계. 공용 인벤토리 분류 피로가 실제 비용
- 스토리 결정이 자원보다 중요해지는 구간
- **체감**: 경제가 플레이를 방해하지 않는 상태 — 설계상 **의도된 완화**

## 8. 경제 UI 표시 원칙

- **자금**: 전 화면 상단 고정 표시. 변동 시 +/− 플로팅 텍스트
- **사기**: 전투 중 전차별 게이지, 정비 씬 통합 뷰
- **파츠 보유량 / 슬롯 점유도**: 정비 씬 좌측 패널
- **"다음 전투 예산" 추천치**: 캠프 진입 시 선택적 튜토리얼 (Phase 2)

**숫자 노출 정책**: 구체 수치는 모두 공개 (블라인드 아님). 사기 밴드·보상 계수 등도 툴팁에 정수·퍼센트 기재.

## 11. 향후 결정 필요 항목

- **전략맵 연료 재도입 여부** (docs/09에서 비포함 결정 — 경제 압박 부족 시 재고)
- **부대 유지비 도입 여부** — 현재 미도입. 중반 공짜 보상 누적 시 도입 검토
- **상점 파츠 SKU 확장 범위** — 주포 판매 허용 여부(노획 경제 훼손 우려)
- **분해 재료 → 업그레이드 비용 테이블** (Phase 2 업그레이드 시스템 도입 시)
- **난이도별 보상 곡선 최종 확정** — 현재 ×1.0/×1.6/×2.4는 초안. 플레이테스트 필요
- **매각가 30% 규칙** — 인플레이션/디플레이션 실측 후 튜닝
- **회식 등급 비용 100/300/800** — 자금 곡선(§6.1) 확정 시 재조정
- **시설 해금 자금 도입 시 금액** (docs/09 §10 항목)
