# 3. 핵심 게임플레이 루프

> 개정: 2026-04-15 — 턴제 부대 단위 전술로 전면 재작성
> 갱신: 2026-04-16 — 부대 인원·자원·분기 수치를 `docs/09` 와 정합성 맞춤
> 이전 판본(실시간 서바이벌 웨이브)은 폐기. 현재는 **Hex 턴제 전술 + 부대 편성** 패러다임.

## 0. 루프 개요

```
          ┌─────────────────────────────────────────┐
          │                                         │
          ▼                                         │
    ┌──────────┐   출격 확정    ┌──────────┐   전투 종료  ┌──────────┐
    │  편성    │ ─────────────▶ │  출격    │ ─────────▶  │  전투    │
    │(Roster)  │               │ (Sortie) │             │(Tactical)│
    └──────────┘               └──────────┘             └──────────┘
          ▲                                                   │
          │                                                   │
          │                                                   ▼
    ┌──────────┐       복귀 정산        ┌──────────────────────────┐
    │  정비    │ ◀─────────────────── │ 복귀 · 노획 (Salvage)    │
    │(Maint.)  │                      │                          │
    └──────────┘                      └──────────────────────────┘
                                                   │
                                                   ▼
                                       (스토리 이벤트 / 맵 해금)
```

| 단계 | 씬 | 한 문장 |
|---|---|---|
| **편성** | Hangar | 부대 전차·승무원·파츠를 조합하고 다음 출격 파티를 고른다 |
| **출격** | StrategyScene | 맵에서 스테이지를 선택하고 파티를 확정한다 |
| **전투** | TacticalScene (Hex) | 턴제 Hex 전술 전투로 목표를 수행한다 |
| **복귀·노획** | Results | 결과 정산, 파츠·자원 노획, 부상 판정 |
| **정비** | Hangar | 수리·스킬 선택·연구 등 다음 출격 준비 |

플레이 한 세션 감각: 편성·정비 5~15분 + 출격 1~2분 + 전투 15~30분. 큰 교전 1회 단위가 30분~1시간 세션의 뼈대.

## 1. 편성 씬 (Roster / Hangar)

### 1.1 부대 운용 단위

- **부대(Convoy)** — 플레이어가 운용하는 전차 단체. **출격 슬롯 3→5 (한 작전 출전 인원), 보관 슬롯 5→10 (게러지 수용)** 단계 확장
- 단계: 초기 출격 3/보관 5 → 중반 1 출격 4/보관 7 → 중반 2 출격 5/보관 8 → 후반 출격 5/보관 10 (상세 `docs/09 §4`)
- 부대 인원 확장은 **스토리 진행**과 **거점 시설(게러지) 해금**에 연동. 임의 구매 없음
- 각 전차는 **독립 ID**를 가지며, 차체만 고정·파츠와 승무원은 교체 가능
- **로시난테는 항상 부대의 일원**이며, 중형 차체 고정 (docs/05 §로시난테 참조). 보관 1슬롯 점유, 출격은 의무 아님 (일부 스토리 미션 제외)

### 1.2 전차 슬롯 작업

전차 1대당 편성 작업:
- **파츠 장착/교체** — 차체에 엔진·포탑·주포·장갑·캐터필러·탄약고·보조장비 조립 (docs/05 §파츠 경제)
- **승무원 배치** — 5개 직책 슬롯에 풀에서 인원 선택 (docs/04 §1)
- **스킬 슬롯 확정** — 각 승무원 4 슬롯(패시브 2 + 액티브 2) 장착 확인
- **탄약 휴대** — 탄약고 용량 한도 내 탄종 배분

편성은 전차 **단위**로 독립 진행. 부대 내 전차 간 파츠·승무원 자유 이동 가능.

### 1.3 이번 출격 파티 선택

- 부대 전체가 자동 출격하지 않음. **플레이어가 이번 스테이지에 내보낼 N대를 선택**
- 선택 제약: 스테이지 난이도·지형·서사 요구에 따라 1~최대 부대 규모
- 비출격 전차는 창고 대기 (부상 승무원 회복, 파츠 정비 진행)

**이유**: 기동력·주포형·수리 특화 같은 **전차 역할 교대**를 플레이어가 전략적으로 계획하게 한다.

## 2. 출격 씬 (Sortie Prep)

### 2.1 스테이지 선택

- 전략 맵에서 해금된 거점·스테이지 선택
- 스테이지별 표시: 지형 유형, 적 구성 힌트, 예상 보상, 스토리 분기 여부
- 일부 스테이지는 시간 제한(스토리 진행도 소모) 또는 선택지 배타 구조

### 2.2 스테이지 정보 미리보기

| 항목 | 표시 여부 |
|---|---|
| 지형 유형 | 공개 (숲·도시·사막 등) |
| 적 대략 규모 | 공개 (소·중·대 + 아이콘) |
| 적 구체 구성 | **블라인드** (정찰 스킬로 해금) |
| 노획 예상 티어 | 공개 |
| 서사 이벤트 존재 | 아이콘 힌트 |

### 2.3 선공 예상 UI

전투 시작 시 각 유닛이 굴릴 **이니셔티브 공식**을 미리보기로 표시:

```
초기화 공식 = React + Morale/5 + 특성 보정 + 차체 속도 + d20
```

출격 확정 전 아군 평균 이니셔티브와 적 평균 이니셔티브를 비교해 **선공 우위** 여부를 플레이어가 판단 가능. d20은 변수로 남지만 기대값은 공개.

### 2.4 파티 확정

- 편성 씬에서 고른 파티를 최종 확인
- 탄약·수리 소모품 충전 상태 확인
- 사기 상태 배너 (마크 5 전차장 탑승 여부 등)
- "출격" 버튼 → 전투 씬 로드

## 3. 전투 씬 (Tactical Battle)

Hex 턴제 전술. 상세 규칙은 `docs/06`·`docs/12` 참조. 여기서는 루프 관점에서만 서술.

### 3.1 턴 구조 — 옵션 B (아군 → 적) + 이니셔티브 미니 게이트

```
전투 시작 → 이니셔티브 굴림 (전차별) → 선공 그룹 결정
  반복:
    아군 턴 (모든 아군 유닛 순차)
    적 턴 (모든 적 유닛 순차)
  종료 조건 도달 시 탈출
```

- 동일 진영 내에서는 플레이어가 행동 순서 자유
- React 미니 이니셔티브 시스템으로 **턴 중 반응 사격·오버워치** 삽입 가능

### 3.2 유닛 1턴 행동

**공용 자원**: 전차 AP 8 (하이브리드 C, docs/04 §7)
- 이동: 셀당 1~2 AP (중량·조종수 마크에 따라)
- 기본 사격: 4~6 AP
- 재장전: 2~3 AP (탄약수 마크·대구경 여부)
- 액티브 스킬: 스킬별 AP + 쿨다운
- 회전·대기: 1 AP

**승무원별 스킬 쿨다운**은 독립. 5명이 같은 턴 스킬 사용 가능.

### 3.3 반응 사격 / 오버워치

- 턴 종료 시 잔여 AP ≥3 & 반응형 스킬 예약 → 적 턴 중 트리거 발동
- 1턴 당 유닛당 반응 발동 **1회**
- 시각 연출: 반응 시퀀스 씬 전환 (`FireActionScene`)

### 3.4 전투 종료 조건

| 결과 | 조건 |
|---|---|
| 승리 | 모든 적 격파 / 목표 지점 도달 / 목표 대상 격파 |
| 패배 | 아군 전차 전멸 / 핵심 전차 격파 / 턴 제한 초과 |
| 후퇴 | 플레이어 의도 후퇴 (일부 스테이지만 허용). 노획률 하락 |

전투 종료 → 결과 화면 → 복귀·노획 단계 진입.

## 4. 복귀 · 노획 (Salvage)

### 4.1 결과 정산

각 아군 유닛별 집계:
- 누적 데미지 · 격파 수 · 모듈 파괴 기여
- 부상 판정 (관통 피격 확률 기반, docs/04 §8)
- 사용 AP·스킬 쿨다운·탄약 소모

승무원별 마크 진행:
- 기여도 배분(격파 ≥60% → 1 카운트, 30~59% → 0.5, 미만 → 무)
- 이중 게이트(킬·참전) 재검사 → 마크 레벨업 이벤트 큐에 적재

### 4.2 노획 시스템

**핵심 원칙**: 적을 **어떻게** 격파했는가가 노획 결과를 결정.

| 격파 방식 | 파츠 노획 확률 | 비고 |
|---|---|---|
| 탄약고 유폭 | **최저** | 내부 손상 극심 — 고철만 |
| 주포 관통 직사 | 낮음 | 차체 훼손 |
| 모듈 집중 파괴 | 중간 | 엔진·포탑 등 특정 파츠 보존 |
| 승무원 사살 (탄두 외 수단·관통 불러) | 높음 | 차체 온전 |
| **승무원 전원 사망 + 차체 기동 불능** | **최고** | 노획 이상적 — 완전체 차체 획득 가능성 |

- "승무원 전원 사망"은 내부 패닉·질식·연소 등 **차체는 살아있지만 내부 인원이 전멸**한 상태를 의미
- 플레이어가 의도적으로 노획 최적화하려면 **탄약고·엔진 직격을 피하고 측면 관통·모듈 파괴**로 압박하는 플레이가 유리
- **로시난테 차체**는 고정이며 노획 대상 아님. 적 중형 전차에서 노획한 차체는 **부대 신규 슬롯**으로 추가 가능

### 4.3 노획 산출 구조

```
노획 파츠 후보 = 적 전차가 장착했던 파츠 리스트
각 파츠별 생존 확률 = base × (격파 방식 계수) × (내부 연쇄 손상 보정)
플레이어 인벤토리로 이동 → 공용 인벤토리 귀속
```

- 각 파츠는 **개별 생존 판정** — 탄약고 유폭이 아니어도 개별 파츠는 파손·소실 가능
- 노획된 파츠는 **부대 공용 인벤토리**에 귀속 (특정 전차 종속 아님)

### 4.4 부상자 처리

- 경상: 자동 — 다음 전투 스탯 -10%, 1 전투 후 회복
- 중상: 정비 씬 대기 → 2~3 전투 이후 복귀
- 치명상: **스토리 이벤트 한정** — 전투 중에는 절대 발생하지 않음

### 4.5 자원·경험 정산

- 스테이지 클리어 자원 (파츠 외 화폐·연료·탄약 보충)
- 승무원 마크 레벨업 큐 → 정비 씬에서 개별 처리 (카드 선택)
- 스토리 이벤트 트리거 플래그 수집

## 5. 정비 (Maintenance)

복귀 후 → 편성 씬으로 돌아오기 전 **필수 정비 단계**. 실제로는 편성 씬과 동일한 Hangar UI에서 작업.

> **구현 상세**: 내구도·모듈 복구 곡선·부상 회복 틱·각성 카드 UX·정비 내 사기 이벤트·UI 레이아웃 상세는 보조 문서 [`03b_maintenance.md`](03b_maintenance.md)로 분리. 본 절은 개요.

### 5.1 차체 수리

- 피격 받은 전차의 파츠별 내구 회복
- 파손 파츠는 **일정 턴 수리 대기** 또는 즉시 교체
- 수리 파츠는 부대 공용 인벤토리에서 소비

### 5.2 승무원 스킬 카드 선택

마크 레벨업 큐에 쌓인 각성 이벤트를 처리:
- 풀 쿼리 → 후보 수에 따라 자동 획득(1개) 또는 카드 선택(2개 이상)
- **블라인드 원칙**: 획득 전까지 스킬 내용 비공개
- 획득 직후 장착 슬롯 수정 가능 (패시브 2 · 액티브 2)

### 5.3 사기·부상 회복

- 사기는 전투 간 **자연 회복**되지 않음. 정비 씬 내 특수 이벤트(휴식·격려 등)로 조정
- 부상자는 자동 관리 — 플레이어는 복귀 예정일만 확인

### 5.4 파츠 조합·연구

- 파츠 **호환성 체크** — 하중·출력·규격 3중 체계 (docs/05 §호환성)
- 미장착 파츠 창고 관리
- 연구: 일정 조건 충족 시 신규 파츠 해금 (MVP 범위 밖, 확장 슬롯)

## 6. 루프 외부 — 스토리·맵 진행

전술 전투 루프 바깥의 거시 진행:

### 6.1 전략 맵 이동

- 월드맵 노드 그래프 위에서 다음 노드로 이동 (`docs/09 §1`)
- **이동 자체에 자원·시간 소모 없음** (식량·연료 자원 시스템 없음, 첫 빌드 단순화 결정)
- 노드 종류: 전투 22 / 대화 15 / 거점 8 / 휴식 6 = 총 51 노드
- 거점 노드 도달 시 캠프 진입 (편성·정비·상점·스킬·회식 등 풀 메뉴)

### 6.2 서사 이벤트

- 주요 NPC 대화
- 캐릭터 관계 이벤트 — 초기 5인 간 관계 변화 (docs/04 참조)
- 스토리 한정 파츠·차체 보상 (로시난테 전용 강화 등)
- **치명상 이벤트** — 특정 서사 분기에서만 발생

### 6.3 맵 해금

- **무분기 일직선 캠페인** — 노드 진행은 한 방향 (분기 없음, `docs/09 §1.3`)
- 한 노드 클리어 → 다음 노드 자동 활성화
- 캠프 시설은 진행도 임계점에서 점진 해금 (초반 상점·게러지 → 중반 정비소·훈련소 → 후반 병원·술집)
- 스토리 엔딩 — 스팀 패키지 판매 구조상 명확한 종결
- 후반 분기 노드 도입 가능성은 향후 결정 (현재 무분기)

## 7. 루프 테스트 기준

### 7.1 MVP 루프 최소 구성

- 편성 씬 디버그 패널 (Phase 1) → 정식 Hangar UI (Phase 2)
- 스테이지 선택 1~3종
- 전투 씬 완성 (현재 `TerrainTestScene` 기반)
- 결과 → 노획 → 정비 → 편성으로 **한 바퀴 돌기** 가능

### 7.2 핵심 재미 포인트 검증

루프가 완성된 후 아래를 체크:
- **편성 의사결정이 유의미한가** — 파츠·승무원 교체가 전투 결과에 체감되나
- **노획이 보상적인가** — "전원 사망 노획" 플레이가 플레이어에게 매력적인가
- **마크 각성이 기대되는가** — 전투 중 "이 전투 끝나면 마크 올라!"가 동기부여가 되나
- **사기·부상이 압박이 되는가** — 무작정 돌격만 하면 손해가 명확한가

네 포인트 중 **하나라도 약하면 해당 시스템 재조정 우선**.
