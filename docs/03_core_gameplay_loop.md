# 3. 핵심 게임플레이 루프

```
[정비/육성] → [스테이지 진입] → [실시간 전투(웨이브)] → [전투 보상] → [정비/육성(반복)]
                                                              ↓
                                                    스토리 이벤트 / 다음 구역 해금
```

## 1. 정비/육성 (스테이지 사이)

### 전차 정비
- 파츠 교체/조립: 차체, 엔진, 포탑, 주포, 캐터필러, 장갑, 탄약고
- 새 전차 조립 가능 (충분한 파츠 수집 시)
- 중량/출력 밸런스 조정
- 탄약 보급

### 승무원 관리
- 스킬 레벨업 (사격, 방호, 기동, 수리)
- 승무원 배치 변경
- 부상 회복

### 자원 관리
- 획득한 파츠/자원 정리
- 연구 (새 파츠 해금)

## 2. 스테이지 진입

- 스토리에 따라 다음 구역으로 이동
- 스테이지별 지형/환경 특성 (도시 폐허, 숲, 사막, 공장 지대 등)
- 스테이지 진입 전 적 정보 미리보기

## 3. 실시간 전투

- 탑다운 뷰에서 전차 1대 직접 조종
- **이동**: WASD 또는 조이스틱
- **포탑 조준**: 마우스 방향 (차체와 독립 회전)
- **사격**: 자동 또는 수동 (장착 무기에 따라)
- 적이 웨이브로 밀려옴
- 웨이브 사이 짧은 숨돌리기
- 보스 웨이브 존재
- 스테이지 클리어 조건: 모든 웨이브 격파 / 목표 지점 도달 / 보스 처치

## 4. 전투 보상

- 파괴한 적에서 파츠/자원 드랍
- 스테이지 클리어 보상
- 스토리 이벤트 트리거

## 5. 전투 중 시스템

### 관통/방호
- 기존 물리 시스템 유지: 입사각, 유효 장갑, 관통/도탄 판정
- 실시간으로 적용 (적의 공격이 장갑에 튕기는 시각 효과)

### 데미지 모델
- 모듈별 피격: 캐터필러 피격 → 기동력 저하, 엔진 피격 → 속도 감소
- 탄약고 피격 → 유폭 위험

### 전장 오브젝트
- 파괴 가능한 구조물
- 적 전진을 지연시키는 장애물
