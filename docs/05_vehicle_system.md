# 5. 전차 시스템

## 핵심 철학

Front Mission처럼 **장착 파츠에 따라 플레이 스타일이 완전히 달라지는** 시스템.
유저의 취향을 반영한 커스터마이징이 핵심.

## 전차 모듈 구성

### 차체 (Hull)
- **한계 하중**: 탑재 가능한 총 중량 결정
- **장갑 슬롯**: 추가 장갑 장착 가능 수
- **차체 내구력(HP)**
- 차체 종류에 따라 기본 형태 결정 (경차량 ~ 중전차)

### 엔진
- **출력**: 높을수록 중장비 상태에서도 빠른 기동
- **연비**: 장거리 스테이지 시 중요
- **중량**: 엔진 자체도 무게가 있음
- 출력/중량 비율 = 실질 기동력

### 포탑
- **회전 속도**: 적을 빠르게 조준할 수 있는지
- **탑재 가능 주포 구경 제한**
- **포탑 구동 장치 슬롯**

### 주포 (메인 무기)
- **구경/타입에 따라 플레이 스타일 결정:**

| 무기 타입 | 특성 | 플레이 스타일 |
|----------|------|-------------|
| 소구경 속사포 | 빠른 연사, 낮은 단발 데미지 | 기관총형, 다수 적 처리 |
| 대구경포 | 느린 장전, 높은 관통/데미지 | 저격형, 단발 고위력 |
| 유탄발사기 | 범위 데미지, 낮은 관통 | 범위형, 밀집 적 처리 |
| 로켓/미사일 | 유도, 높은 데미지, 제한 탄약 | 보스 킬러 |
| 화염방사기 | 근거리 지속 데미지 | 근접 돌격형 |

### 캐터필러
- **기동성**: 회전 속도, 가속력
- 파손 시 기동력 대폭 저하

### 장갑
- 추가 장갑 장착 → 방호력 상승, 중량 증가
- 전면/측면/후면 개별 장착 가능

### 탄약고
- 탄약 보유량 결정
- 피격 시 유폭 위험 (크리티컬 리스크)

### 보조 장비 슬롯
- 연막탄 발사기, 조준 장치, 통신 장비 등
- (기획 보강 필요)

## 중량/출력 시스템

```
실질 기동력 = 엔진 출력 / 총 중량
총 중량 = 차체 + 엔진 + 포탑 + 주포 + 장갑 + 탄약 + 보조장비

한계 하중 초과 시:
- 기동력 대폭 저하
- 캐터필러 내구도 가속 소모
- 엔진 과열 위험
```

## 플레이 스타일 예시

| 빌드 | 구성 | 특징 |
|------|------|------|
| **속공형** | 경차체 + 강엔진 + 속사포 + 경장갑 | 빠른 기동, 회피 중심, 체력 낮음 |
| **중전차형** | 중차체 + 대구경포 + 중장갑 | 느리지만 한 방이 강하고 단단함 |
| **범위형** | 중차체 + 유탄발사기 + 중장갑 | 다수 적 처리에 특화 |
| **돌격형** | 중차체 + 화염방사기 + 중장갑 | 근접전 특화, 리스크 높음 |
| **저격형** | 경차체 + 대구경포 + 경장갑 | 원거리 고위력, 기동으로 회피 |
