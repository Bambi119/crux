# 4. 승무원 시스템

> 보병 시스템은 제거. 전차 1대 + 승무원에 집중.

## 승무원 구성

전차 1대에 최대 5명의 승무원 배치:

| 직업 | 역할 | 영향 스탯 |
|------|------|----------|
| **전차장** | 전체 보너스 + 특기(패시브 스킬) | 모든 스탯에 소폭 보너스 |
| **포수** | 주포 사격 정확도, 조준 속도 | 명중률, 조준 시간 감소, 크리티컬 확률 |
| **탄약수** | 장전 속도, 탄종 교환 | 연사율 상승, 탄약 효율 |
| **조종수** | 전차 기동성 | 가속, 회전, 최고 속도 보너스 |
| **기총사수/수리병** | 보조 무기 운용 + 모듈 수리 | 보조 무기 명중률, 수리 속도 |

## 승무원 스탯

| 스탯 | 설명 |
|------|------|
| **숙련도** | 직업별 전문 능력 (위 직업별 영향 참조) |
| **체력** | 승무원 개인 HP — 부상 시 효율 저하 |
| **정신력** | 공포/패닉 저항 — 낮으면 성능 저하 |

## 성장

- 전투 경험치로 숙련도 상승
- 스킬 트리: 직업별 특화 스킬 선택
  - 예) 포수: "정밀 사격" vs "속사" 분기
  - 예) 조종수: "드리프트" vs "안정 주행" 분기
- 전차장 특기는 고유 패시브 (캐릭터마다 다름)

## 전차장 특기 예시

| 특기 | 효과 |
|------|------|
| 정밀 지휘 | 명중률 +5%, 조준 시간 -10% |
| 강철 의지 | 전 승무원 정신력 +20% |
| 전장의 눈 | 시야 범위 +15% |
| 응급 정비 | 모듈 수리 속도 +30% |
