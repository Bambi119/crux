## 1. 설계 원칙

1. **3축 분리** — 캐릭터·전차·부대는 서로 다른 시간 척도와 영속성으로 성장. 한 축의 손실이 다른 축으로 보전되지 않음 (영구 사망한 캐릭터의 마크는 자금으로 환산되지 않음)
2. **명시 진행 vs 묵시 진행** — 캐릭터 마크는 진행도 가시(`docs/04 §3.4`), 전차/부대는 인벤토리·자금 직접 표시. 숨김 게이지 금지
3. **외부 회복 경로 보장** — 영구 사망·차체 파괴는 게임 종료 위협. 신규 충원·노획·자금 수입 3대 회복 경로가 항상 활성
4. **선형 확장이 아닌 사다리 확장** — 파츠 등급·마크·차체 등급 모두 5단계 사다리. Act 진행과 동기화
5. **루프 폐곡 의무** — 한 사이클(편성→전투→정비→편성) 내에서 모든 성장 축이 최소 1회 이상 표시 변화 가능해야 함

## 2. 3축 성장 개관

| 축 | 단위 | 영속성 | 권위 문서 | 진행 트리거 |
|---|---|---|---|---|
| 캐릭터 | 승무원 1명 | 사망 시 소실 | `docs/04 §3` 마크, `docs/04b §4` 스케일링 | 직책별 axis 누적 (킬·전투) |
| 전차 | 차체 1대 | 파괴 시 차체 소실, 파츠는 부분 노획 | `docs/05 §4·§5` 파츠 경제 | 노획·구매·분해→조합 |
| 부대 | 캠페인 1개 | 캠페인 종료까지 영속 | `docs/11` (미작성, 본 문서 §5 임시 정의) | 인원 충원·차체 노획·자금 수입 |

**상호 의존**:
- 캐릭터 성장은 탑승 전차의 호환성으로 발현 (`docs/04b §3.2` 호환성 3상태)
- 전차 성장은 정비 인원·자금 필요 (`docs/03b §3` 자동 처리)
- 부대 성장은 출격 가능 차체 수 = 캐릭터 직책 충원 가능성에 직결

## 2.5 3축 성장 사슬 (Progression Chain)

각 축의 **행동 → 조건 → 결과** 인과를 한 페이지에 보존. 메커니즘 권위는 04/05/05b로 이관, 본 절은 흐름 다이어그램만.

### 2.5.1 캐릭터 성장 사슬

```
[행동] 전투 (격파 / 탑승 / 수리 / 사기 이벤트)
   │
   ├──────────── 마크 트랙 ────────────┐
   │                                    │
   │   [축 카운트] 공통 axis (성장 축)   │
   │   탑승 전차의 적 격파 1건당         │
   │   차내 승무원 5명 모두 +1           │
   │   (직책 무관, 차 단위 단일 카운트)  │
   │                                    │
   │   [조건] 이중 게이트 (docs/04 §3.3)│
   │   kills ≥ killThreshold[N]         │
   │     [1, 5, 15, 35, 70] ⚠ 잠정      │
   │   AND                              │
   │   battles ≥ battleThreshold[N]     │
   │     [1, 2, 5, 10, 20] ⚠ 잠정       │
   │                                    │
   │   [이벤트] 마크업                   │
   │   → AwakeningQueue (각성 대기열)   │
   │                                    │
   │   [UX] 정비 탭 각성 카드 (블라인드) │
   │   직책별 풀에서만 추출:             │
   │   ├─ 발동형 직책 (포수·탄약수·     │
   │   │   조종수): 강화 vs 신규 1장   │
   │   └─ 패시브·액티브 (전차장·수리):  │
   │       신규 카드 1장                │
   │                                    │
   │   [결과] 마크 보너스                │
   │   +3 / +6 / +10 / +14 / +20        │
   │   직책별 효과로 발현 (docs/04 §3.2)│
   │   + 카드 효과 (호환성 통과 시 활성)│
   │                                    │
   └──────────── 트레잇 트랙 ─────────┐
                                       │
       [축 카운트] 트레잇 axis        │
       (캐릭터 고유, 마크와 독립)     │
       예: 티티 = 탑승 전차 유폭      │
                                       │
       [조건] 캐릭터별 임계값          │
       예: 유폭 10회                   │
                                       │
       [이벤트] 트레잇 활성            │
       → 정비 탭 트레잇 슬롯 알림     │
                                       │
       [결과] 고유 스킬 자동 장착      │
       (직책 슬롯 4개와 별개 소켓)     │
       + 그라인드(반복 플레이) 동기   │
```

권위: `docs/04 §3.1~3.5` (axis·조건·트레잇) · `docs/04b §4` (카드 분기·강화 스택)

### 2.5.2 전차 성장 사슬

```
[행동] 적 격파
   │
   ▼
[조건] 격파 방식 판정 (docs/05 §5.1)
   탄약고 유폭 → 쓰레기 / 주포 직격 → 일반 / 모듈 집중 → 숙련 /
   승무원 사살 → 정예 / 승무원 전원 사망 → 전설 (스토리)
   │
   ▼
[산출] 노획 알고리즘 (docs/05b §3~5)
   격파 방식 계수 × 카테고리별 베이스 생존률 → 파츠 산출
   인접 폭발 보정 / 등급 변환 / 차체 노획 별도 판정
   │
   ▼
[입고] 공용 인벤토리 (ConvoyInventory)
   │
   ▼
[장착 시도] 정비 탭 즉시 교체 / 격납고 편성 탭
   │
   ▼
[조건] 호환성 3중 체계 (docs/05 §3)
   하중(weight) ≤ 차체 한도 AND 출력(power) ≤ 엔진 여유 AND 규격(spec) 일치
   │
   ▼
[결과] 카테고리별 능력 변화 (§4.2.1 등급 결과치 표)
   숙련 +15% / 정예 +35% / 전설 고유 효과
   │
   ▼
[피드백] 더 좋은 파츠 → 더 정밀한 격파 → 더 좋은 등급 노획 (자기강화 루프)
```

권위: `docs/05 §3·§4` (등급/호환성) · `docs/05b §3~5` (노획 산출)

### 2.5.3 부대 성장 사슬

```
[행동] 전투 종료
   │
   ▼
[수입] 자금 보상 + 노획 차체 + 인벤토리 누적 (격파 등급별)
   │
   ▼
[갱신] 격납고 진입 → 인벤토리 정리 / 차체 풀 갱신 / 캠프 방문 카운트 ++
   │
   ▼
[조건] 출격 가능 차체 수 변동
   Act 1: 1~2대 / Act 2: 3~4대 / Act 3: 5~6대 (상한 6)
   │
   ▼
[이벤트] 신규 직책 충원 압력
   ├─ 스토리 합류 (Act 진행 중 정해진 시점, §11.2)
   ├─ 추모식 (영구 사망 발생 시, docs/03b §5)
   └─ 절차 생성 모집 (Phase 2)
   │
   ▼
[결과] 부대 4 자원 정량 변화 (§5.5 부대 규모별 정량 표)
   총 AP/턴 ↑ · 전선 폭 ↑ · 동시 사격 ↑ · 1 전투 보상 ↑
   │
   ▼
[피드백] 부대 폭 확장 → 1 전투당 누적 보상 ↑ → 정비 부담 ↑ → 다음 충원 압력
```

권위: 본 §5 (임시) → `docs/11 경제 시스템` 작성 시 이관

## 3. 캐릭터 성장 — 마크 진행 페이스

### 3.1 권위 위치

- **마크 수치 효과**: `docs/04 §3.2` (Mark 1~5: +3/+6/+10/+14/+20)
- **이중 게이트**: `docs/04 §3.3` (killThreshold·battleThreshold)
- **스킬 분기**: `docs/04b §4` (발동형 강화 vs 패시브·액티브 카드)
- **카드 UX**: `docs/03b §4.1` (정비 탭 각성 카드)

본 절은 **위 메커니즘 위에서의 실제 진행 페이스**만 정의.

### 3.2 평균 도달 전투 수 — 시뮬레이션

`docs/04 §3.3` 게이트 기준. 격파 기여 60%↑를 풀 카운트로 가정. 부대 출격 4대(중반), 전투당 격파 5~7대 평균 가정.

| 마크 | killThreshold | battleThreshold | 평균 전투 도달 (포수 기준) |
|---|---|---|---|
| Mark 1 | 1 | 1 | 1전투 |
| Mark 2 | 5 | 2 | 3~5전투 |
| Mark 3 | 15 | 5 | 8~15전투 |
| Mark 4 | 35 | 10 | 20~30전투 |
| Mark 5 | 70 | 20 | 40~60전투 |

**해석**:
- Act 1 (1~10 전투): Mark 2~3 도달
- Act 2 (11~30 전투): Mark 3~4 도달
- Act 3 (31~60 전투): Mark 4~5 도달
- 캠페인 1회 클리어 ≈ 50~80 전투 → 핵심 직책 1~2명 Mark 5 도달 가능

### 3.3 공통 axis (성장 축) — 2026-04-22 갱신

> **변경**: 직책별 axis 분리 폐지. 공통 axis 단일 트랙으로 통합 (`docs/04 §3.1`).

- **누적 규칙** — 탑승 전차의 적 격파 1건당 차내 승무원 5명 모두 +1
- **차 단위 단일 카운트** — 차내 기여도 분리 없음. 다른 차의 승무원은 0
- **직책별 마크 효과 분기는 유지** — axis는 공통, 마크업 보너스는 직책마다 다르게 적용 (`docs/04 §3.2`)
- **직책별 카드 풀 분기는 유지** — 마크업 추출 카드는 직책 풀에서만 (`docs/04b §1.2`)

> **차 간 양보 메타 = 의도** — 강한 적은 숙련 부대가, 약한 적은 신규 충원 부대가 정리. 차 간 표적 선택의 전략 자유도로 활용.

> **⚠️ 임계값 재튜닝 잠정** — §3.2 평균 전투 수는 직책별 axis 분리 시절(보직마다 카운트 분산)에 산정. 공통 axis 채택 후 전원이 동시에 카운트를 받으므로 **마크 5 도달 페이스가 너무 빠를 가능성**. 정확한 재튜닝은 `docs/09 맵 & 스테이지` Act별 적 등장 수 확정 후 (§11 오픈 이슈 추가).

#### 3.3.1 트레잇 트랙 (별개)

마크 axis와 별도로 캐릭터별 **트레잇 axis (고유 성장 축)** 가 병행 진행. 고유 스킬 1개를 잠금 해제. 직책 슬롯 4개와 별개 소켓에 장착. 상세는 `docs/04 §3.5`.

**의미**: 한 캐릭터가 다양한 환경에 노출될수록 여러 axis가 진행 → 빌드 다양성. 단일 환경 고정은 1개 axis만 빠르게 Mark 5.

### 3.4 가시화 정책

- 진행도 표시: 격납고 승무원 카드 + 정비 탭 각성 카드 진입
- 다음 카드 효과: **블라인드** (`docs/04 §3.4`)
- 마크업 알림: 전투 종료 결과 화면 → 정비 탭 각성 큐로 전달 (`docs/03b §4.1.1`)

### 3.5 마크 결과 시나리오 — Before / After

권위 표(`docs/04 §3.2`)의 단일 수치 +3/+6/+10/+14/+20을 실제 게임플레이 효과로 환산.

#### 시나리오 A — 포수 격파 효율 (대구경 axis)

| 항목 | Mark 1 포수 | Mark 5 포수 |
|---|---|---|
| axis 보너스 | +3 (명중) | +20 (명중) |
| 기본 명중률 60%, 평지 사거리 4 | **63% 명중** | **80% 명중** |
| 격파 시도 5회/전투 | 평균 3.15격파 | 평균 4.0격파 |
| 격파 정밀도 (모듈 적중률) | 보통 | 높음 → 정예 파츠 노획 빈도 ↑ |

→ **단순 DPS 1.27배**, 노획 등급 영향까지 합치면 부대 자원 처리량 **체감 1.5~1.8배**.

#### 시나리오 B — 회피 기동 (조종수 발동형 강화 스택, `docs/04b §4.1.1`)

| 항목 | Mark 1 | Mark 5 |
|---|---|---|
| 발동률 | 15% | 55% |
| 적용 범위 | 정면 피격만 | 전 방위 피격 |
| 가정: 전투당 피격 8회 (정면 3·측면 3·후면 2) | 0.45회 발동 | 4.4회 발동 |

→ **체감 생존성 ≈ 9.8배** 발동 빈도. 회피 1회당 받는 피해 -50% 가정 시 누적 받피해 약 **-40%**.

#### 종합 해석

- 마크 1→5의 정량 차이 = **+1.3배 ~ 10배** (직책별·스킬별 분산)
- **axis 매칭 환경에서만 발현** — 환경 다양성이 부대 효율 결정
- 발동형 스킬 강화 스택의 효과량이 패시브·액티브 신규 카드보다 크게 체감 (마크 5 회피 기동 vs 마크 5 정밀 조준 +5%)

## 4. 전차 성장 — 파츠 사다리

### 4.1 권위 위치

- **파츠 등급 5단계**: `docs/05 §4.3` (쓰레기·일반·숙련·정예·전설)
- **격파 방식별 등급 분포**: `docs/05 §4.3` 표

- **호환성 3중 체계**: `docs/05 §3` (하중·출력·규격)

### 4.2 성장 곡선 — 평균 파츠 등급

격파 방식이 등급을 결정하므로, 플레이어 전술 숙련도와 직접 연동.

| Act | 평균 보유 파츠 등급 | 배경 |
|---|---|---|
| Act 1 | 쓰레기·일반 | 시작 5인 부대, 단독·2대 출격 — 격파 정밀도 낮음 |
| Act 2 | 일반·숙련 | 4대 부대, 모듈 사격 가능 — 등급 상승 |
| Act 3 | 숙련·정예 | 6대 부대, 캐터필러 무력화·승무원 사살 전술 정착 |
| 후반 거점·이벤트 | 정예·전설 | 승무원 전원 사망·스토리 보상 |

**플레이어 압력**: 더 좋은 파츠 = 더 정밀한 격파 = 더 좋은 파츠. **게임 외부 학습 곡선**과 일치.

### 4.2.1 파츠 등급별 결과치 (잠정)

> ⚠️ **권위 이관 대상**: 본 표는 `docs/05 §4.3` "+10~20% / +30~40%" 범위 명시를 정량화한 잠정안. `docs/05 §11` "파츠 등급별 보너스 수치 (현재 초안)" 결정 시 본 표 → 05로 이관, 07은 포인터화.

#### 표준 룰

- **숙련 +15% / 정예 +35% / 전설 고유 효과** (모든 카테고리 일관)
- **캐터필러만 +20% / +40%** (소모성·내구 의존성으로 편차 큼)
- **쓰레기 페널티 -10%** (캐터필러는 -20%) — 호환성 + 인벤토리 압박 시 사용

#### 카테고리별 결과치

| 카테고리 | 쓰레기 | 일반 | 숙련 | 정예 | 전설 (예시) |
|---|---|---|---|---|---|
| 엔진 출력 (Power) | -10% | 기준 | +15% | +35% | 고유 — 무열(연료·열 무시) |
| 주포 관통 (Penetration) | -10% | 기준 | +15% | +35% | 고유 — 항상 관통 1발/전투 |
| 주포 명중 (Accuracy) | -10% | 기준 | +15% | +35% | 고유 — 도탄 무시 |
| 장갑 두께 (Armor) | -10% | 기준 | +15% | +35% | 고유 — HEAT 면역 |
| 탄약고 용량 (Ammo) | -10% | 기준 | +15% | +35% | 고유 — 유폭 면역 |
| 캐터필러 내구 (Track) | -20% | 기준 | +20% | +40% | 고유 — 지형 페널티 무시 |
| 보조 장비 효과 (Auxiliary) | -10% | 기준 | +15% | +35% | 고유 — 사거리/지속 무한화 |

#### 전설 등급 정책

- **정량 보너스 대신 게임플레이 변화 효과** — "+50% 관통" 같은 단순 인플레 회피
- **빈도**: 캠페인당 0~3개 (스토리 보상 + 전원 사망 노획 한정, `docs/05 §5.2`)
- **Phase 1 MVP**: 1~2개 정의 (예: 무열 엔진, HEAT 면역 장갑)
- **호환성 우회 금지** — 전설도 호환성 3중 체계 통과 필수

#### 정합성 가드

- 부대 평균 등급이 §6 권장 분포 표와 ±1단계 이상 괴리 시 **노획률 표 조정** (`docs/05b §3`)
- 등급별 보너스는 차체 베이스 스탯 곱연산 — 누적 곱셈으로 인한 폭주 방지: 동일 카테고리 1슬롯/차체 (`docs/05 §1.1` 슬롯 구조 준수)

### 4.3 차체 등급 사다리

`docs/05 §1` 5종 차체 (경·중·중량·SPG·로시난테). Act 진행에 따라:

- Act 1: 로시난테 + 노획 차체 1~2대 (경·중)
- Act 2: 4대 부대 (경 1, 중 2, 로시난테)
- Act 3: 6대 부대 (중·중량 다양 조합 가능)

차체 자체는 노획·이벤트 외 업그레이드 없음 (모듈 교체로 성능 변화).

### 4.4 부분 분해 경로

`docs/05 §4.4` 분해→소재 환원→같은 카테고리 수리·업그레이드 사용. 등급 하락 교환 비율 (정예 1 → 숙련 재료 ~2). **본 문서는 정책만 인용** — 분해 UI는 격납고 모듈 (`docs/10b`) 후속 작업.

## 5. 부대 성장 (Convoy Progression)

> **임시 권위**: `docs/11 경제 시스템` 미작성 상태. 본 절이 부대 성장 골격을 임시 보유. 11 작성 시 본 절은 포인터화.

### 5.1 부대 4 자원

| 자원 | 단위 | 변동 트리거 | 권위 위치 |
|---|---|---|---|
| 인원 (Crew) | 명 | 충원 이벤트·사망 | 본 절 §5.2 |
| 차체 (Tank) | 대 | 노획 (`docs/05 §5.4`)·파괴 | 본 절 §5.3 |
| 자금 (Credit) | 정수 | 전투 보상·판매·정비 비용 | 본 절 §5.4 |
| 공용 인벤토리 | 파츠·소재 | 노획·분해·구매 | `docs/05 §4.1` |

### 5.2 인원 충원 경로 (잠정)

1. 시작 5인 (`docs/04 §9.2`)
2. 스토리 합류 — Act 진행 중 정해진 시점에 1~2명 (의사 등 — `docs/03b §11 오픈 이슈`)
3. 절차 생성 모집 풀 — Phase 2 (`docs/04 §9.3`)

**현재 MVP 가정**: 시작 5인 + 스토리 합류 ~3명 = 캠페인 풀 8명 안쪽.

### 5.3 차체 보유 상한

- Act 1: 1~2대 (단독 출격 → 2대 부대)
- Act 2: 3~4대
- Act 3: 5~6대 (출격 상한 6대)

**상한 초과 시 처리**: 노획 차체 인벤토리 보관 (출격 미선택 가능). 격납고 슬롯 한계는 미정 (오픈 이슈 §11.4).

### 5.4 자금 흐름 (잠정 — 11 본권위 시 이관)

| 수입 | 지출 |
|---|---|
| 전투 보상 (격파 등급별) | 정비 (현재 무료, `docs/03b §1` — 11에서 재검토) |
| 파츠 판매 (분해 외 직판) | 파츠 구매 (상점 — 미정) |
| 스토리 보상 | 인원 모집 비용 (Phase 2) |

**MVP 단순화**: 자금은 결과 화면 표시 + 정비 무료. 상점·구매는 Phase 2.

### 5.5 부대 규모별 게임플레이 정량 변화

부대 규모(출격 차체 수)에 따른 1 전투 사이클 정량 효과. 부대 성장이 게임플레이에 어떻게 발현되는지 가시화.

| 부대 규모 | 총 AP/턴 | 전선 폭 (Hex) | 동시 사격 | 1 전투 평균 보상 (자금/파츠) | 정비 부담 (캠프당) |
|---|---|---|---|---|---|
| 1대 (Act 1 초반) | ~10 | 1 | 1발 | +50 / 1~2개 | 1대 분량 (~3 캠프) |
| 2대 (Act 1 중반) | ~20 | 2~3 | 2발 | +100 / 2~4개 | 2대 분량 |
| 4대 (Act 2) | ~40 | 4~5 | 4발 | +250 / 5~8개 | 4대 분량 |
| 6대 (Act 3 풀스펙) | ~60 | 6~8 | 6발 | +400 / 10~15개 | 6대 분량 (~5 캠프) |

**가정**:
- 차체당 평균 AP 10/턴 (`docs/04 §7` AP 하이브리드 C 기준)
- 전선 폭은 8×10 hex 그리드 기준 동시 점유 가능 hex 폭
- 보상은 §5.4 자금 흐름 + §4.2 평균 등급 기반 — 격파 등급별 보너스 누적

**관찰**:
- 부대 규모 ×2 → 보상 ×2~2.5 (**초선형 — 격파 정밀도 향상이 누적**)
- 정비 부담은 **선형** (대수에 비례)
- 6대 풀스펙 도달 후: 자원 처리량 vs 정비 UI 부담 균형이 깨질 수 있음 → §11.7 가시화 강화 후속

**부대 직책 분배 압력** (6대 시점):
- 직책 5종 × 6대 = **30 슬롯**, 시작 5인 + 충원 ~3명 = 8명 → 슬롯 27개 공석
- 공석 = 출격 불가 (`docs/04 §8.1` Hard Gate) → **충원 페이스가 부대 성장 천장**

## 6. 진행 페이스 권장치

3축 성장이 동기화되어야 난이도 곡선이 매끄러움. Act별 권장 분포:

| Act | 평균 마크 | 평균 파츠 | 부대 규모 | 전형 전투 수 |
|---|---|---|---|---|
| Act 1 (튜토리얼) | 1~2 | 쓰레기·일반 | 1~2대 | 1~10 |
| Act 2 (전개) | 2~3 | 일반·숙련 | 3~4대 | 11~30 |
| Act 3 (절정) | 3~4 | 숙련·정예 | 5~6대 | 31~60 |
| 결말 | 4~5 (핵심 1~2명) | 정예·전설 (포인트) | 6대 풀스펙 | 60+ |

**검증 도구**: §10 튜닝 훅으로 게이트·노획률 조정. 60전투 클리어 시 핵심 직책 Mark 5 도달이 안 되면 게이트 완화.

## 7. 영구 사망과 손실 회복

### 7.1 영구 사망 정책

`docs/04 §8` 부상 시스템 + `docs/03b §3.3` 부상 회복 (3전투 자연 회복) + 치명상 정책 (`docs/04 §8.2`).

**손실되는 것**:
- 승무원 1명 (마크·스킬·특성 전부)
- 사기 -15 즉시 (`docs/04 §6.1`)
- 직책 공석 발생 시 출격 불가 (Hard Gate, `docs/04 §8.1`)

**손실되지 않는 것**:
- 자금·파츠·차체
- 다른 승무원의 마크 (별 단위 누적이라 환산 불가)

### 7.2 회복 경로

1. **충원 이벤트** — 스토리·절차 생성으로 신규 가입 (Mark 0 또는 낮은 마크)
2. **추모식** — 정비 탭 사기 이벤트 +5 (`docs/03b §5`)
3. **빌드 재구성** — 잃은 직책의 다른 캐릭터를 그 차체에 배치

**시스템 안전장치**: 시작 5인 중 핵심 직책(전차장·포수·조종수) 1명만 남는 상황 = 사실상 게임 오버. 이 문턱 이전에 충원 이벤트가 트리거 되어야 함 — Act 진행과 동기화 (오픈 이슈 §11.2).

### 7.3 차체 파괴와 회복

차체 파괴 = 출격 차체 1대 즉시 소실. 모듈은 부분 노획 가능 (격파 방식 = 우리 측 격파니까 어떻게 산정? 오픈 이슈 §11.3).

회복: 노획 차체 + 신규 차체 이벤트.
