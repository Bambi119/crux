# 6. 전투 시스템

## 전투 방식

**실시간 탑다운 서바이벌** — 뱀서류 기반이되, 전차 특성을 살린 전투

### 조작
- **WASD**: 차체 이동 (방향키 대응)
- **마우스**: 포탑 조준 (차체와 독립 회전)
- **좌클릭/자동**: 주포 사격
- **우클릭**: 보조 무기/스킬
- **Shift**: 긴급 기동 (엔진 부스트, 쿨다운 있음)

### 전투 흐름
1. 스테이지 시작 → 전차 배치
2. 웨이브 시작 → 적이 사방에서 밀려옴
3. 웨이브 클리어 → 짧은 휴식 (드랍 아이템 수집)
4. 다음 웨이브 → 적 강도 증가
5. 최종 웨이브/보스 → 클리어

## 관통/방호 시스템

실시간이지만 **관통 판정은 유지**:

```
적 공격 → 명중 → 입사각 계산 → 유효 장갑 비교 → 관통/도탄
```

- 도탄 시: 파편 파티클 + "땡!" 효과음
- 관통 시: 내부 데미지 + 모듈 손상 가능

### 유효 장갑 계산 (기존 유지)
```
유효 장갑 = 기본 장갑 / cos(입사각)
입사각 70도 이상 → 자동 도탄
```

### 모듈 피격
- **캐터필러**: 기동력 -50%
- **엔진**: 최대 속도 -30%, 과열 위험
- **포탑 구동장치**: 포탑 회전 속도 -50%
- **탄약고**: 유폭 판정 (HP 대량 손실 또는 즉사)
- 승무원 수리 스킬로 전투 중 복구 가능 (시간 소요)

## 적 유형

### 경량 기계
- 빠르고 약함, 대량 출현
- 뱀서의 일반 몹 역할

### 중장갑 기계
- 느리고 단단함, 관통이 어려움
- 측면/후면 공략 필요

### 원거리 기계
- 멀리서 사격, 회피 기동 필요

### 폭발 기계
- 근접 시 자폭, 광역 데미지

### 보스 기계
- 스테이지 최종 웨이브
- 패턴 공략 필요
- 바이오 코어 탑재

## 전장 환경

### 지형 효과
| 지형 | 효과 |
|------|------|
| 아스팔트 | 기본 |
| 진흙 | 이동 감속 |
| 잔해 | 파괴 가능, 시야 차단 |
| 물웅덩이 | 감속 + 엔진 과열 방지 |

### 파괴 가능 오브젝트
- 건물 잔해, 바리케이드, 차량 잔해
- 파괴 시 파편 파티클
- 일부는 적에게도 장애물 역할
